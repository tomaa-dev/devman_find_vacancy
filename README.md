# Сравниваем вакансии программистов
Данный скрипт представляет собой консольное приложение на Python, которое скачивает вакансии разработчиков с HeadHunter и SuperJob. Утилита позволяет пользователю проанализировать и сравнить среднюю зарплату для каждого языка программирования на вышеуказанных сайтах в виде сводных таблиц. 

## Как установить
Операционная система: любая, где доступен Python 3 (Windows, macOS, Linux). Используем 'pip' для установки зависимостей (requests, python-dotenv):
```
pip install -r requirements.txt
```
Рекомендуется использовать virtualenv/env для изоляции проекта.

## Примеры запуска скриптов

Вывод вакансий в виде таблиц осуществляется следующим образом:

```
python find_job.py
```

## Переменные окружения
Программа использует нестандартные переменные окружения для настройки. Эти переменные не могут быть автоматически обнаружены и должны быть указаны в файле `.env` перед запуском программы.

**SUPER_JOB_API_KEY** - токен доступа к данным API SuperJob, необходимый для авторизации. Форматом является строка (например, `v3.r.139481774.b21af38dfc4e6eec163bd9928704d0a2b0dda5e7.e5c160cf9837a9dc9c269515aeb3e56af9fa9728`). Без этого ключа сайт не будет работать, и программа завершит выполнение с ошибкой.
